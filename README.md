# Student-performance-prediction
A machine learning web application that predicts student academic performance using Flask and Scikit-learn.

# Overview
Student Performance Prediction is a machine learning web application that predicts student's academic performance based on demographic and academic information. The application is built using Flask and Scikit-learn.

# Tech Stack
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Joblib
- HTML
- CSS
  Dataset

# The project uses the **Students Performance Dataset**, which includes features such as:
- Gender
- Race/Ethnicity
- Parental level of education
- Lunch
- Test preparation course
- Math score
- Reading score
- Writing score
